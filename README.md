# Mini-Supermarket-Sales--Analysis
Sales analysis using Excel (pivot tables, charts and basic insights)

## 📊 Project Overview
This project analyzes a small supermarket dataset to understand sales performance, product demand, and customer behavior.

## 🧾 Dataset
- Dataset was manually created for practice purposes
- Contains 30 transactions including product, price, quantity, and customer data

## ❓ Questions Answered
- What is the total sales?
- Which product generated the most revenue?
- Which product sold the highest quantity?
- Who is the top customer?
- What is the average transaction value?

## 📈 Key Insights
- Best Product (Revenue): Sugar
- Most Sold Product: Rice
- Top Customer: Zainab
- Total Sales: ₦142,500
